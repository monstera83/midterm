12223769 이상원 오픈소스 중간고사 과제

u.item, u.data, u.user 를 itemFile, dataFile, userFile로 받아 변수명으로 프로그램에서 호출한다.

1번 문제
사용자에게 영화 번호를 입력받아 u.item에서 awk로 해당 아이디의 영화의 정보를 조회함
파이프로 정보가 구분되어있어서 -F\|로 구분자를 바꿈.

2번 문제
u.item에서 awk로 action장르가 1인 값만 출력함.
출력한 값을 한 번 더 awk로 가장 첫 10개의 값만 출력.

3번 문제
u.data에서 사용자에게 입력받은 번호의 영화의 평점을 모두 더하고 마지막에 개수로 나누어 주어 평균을 구함.

4번 문제
url정보를 제거하고 출력해야 하기 때문에 awk로 해당 번호빼고 모두 출력해준다.

5번 문제
u.user에서 사용자에게 입력받은 번호를 검색, 찾아서 순서를 맞춰서 배열한다. M 과 F는 sed를 통해 각각 Male과 Female로 변경함.

6번 문제
u.item에서 1673~1682 사이의 값을 가져오고 sed를 통해 (숫자두자리)-(영문자세자리)-(숫자네자리)를 (숫자네자리)(영문자세자리)(숫자두자리)로 배치를 바꾸고
영문자에 해당하는 숫자는 각각 따로 처리해주었다.

7번 문제
u.data에서 받은 번호로 본 영화를 조회하고 영화번호만 따로 숫자정렬하여 | 를 중간에 끼워주었다.

9번 문제
프로그램은 while문 안에서 계속 반복되고 9를 입력받으면 종료된다.
